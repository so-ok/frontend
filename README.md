## 영양 쏘옥

---

코로나로 인해 생활 속에서 건강을 관리하고 투자하는 ‘셀프 메디케이션’이 트렌드로 떠오르고 있습니다.

영양 쏘옥 에서는 영양제를 성분, 이름, 관심 분야의 항목으로 간편하게 찾고 사용자가 복용하고 있는 영양제의 성분을 일일 권장 섭취량의 최소치 최대치를 간편하게 보여주고 추천하는 서비스입니다.
